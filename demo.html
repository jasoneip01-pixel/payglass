🚀 运行 Demo

git clone https://github.com/jasoneip01-pixel/payglass.git
cd payglass
# 8 scenarios, 3 tracks, 1 validation catch
python3 demo.py

# View the live dashboard
python3 -m http.server 8080 -d dashboard/
# → http://localhost:8080

运行后 demo.py 会自动生成 dashboard/data.json，Dashboard 页面会自动加载最新数据。
